            • java.util.List<java.lang.String> multiFeatureNames(java.util.Map args)

              Given some data, a subject ID key, and a feature name, construct a list of multi-feature names that cover the number of discrete

              Parameters:
              args.data - The input data that must contain the subject ID key.
              args.subjectIdKey - The key of args.data that contains the subject IDs.
              args.featureName - The name used as a prefix for constructed multi- feature names.
              Returns:
              A list of multi-feature names. Data example: EMPI TUMOR_ID OTHER_FEATURE 1 1 a 1 2 b 2 3 c 3 4 d 4 e 4 f 4 g Example call: multiFeatureNames( data, subjectIdKey:'EMPI', featureValueKeys:['TUMOR_ID'] featureName:'SEVERITY', ) Results: ['SEVERITY_1', 'SEVERITY_2'] Explanation: EMPI 1 has two TUMOR_ID values. All other EMPIs have 0 or 1 vaues. So, the max number of severities in these data is 2 and we have fields SEVERITY_1 and SEVERITY_2 to contain them.
