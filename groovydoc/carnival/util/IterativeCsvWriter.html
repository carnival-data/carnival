Package: carnival.util

[Groovy] Class IterativeCsvWriter

    • carnival.util.IterativeCsvWriter 

    • @groovy.util.logging.Slf4j
class IterativeCsvWriter
extends java.lang.Object

      IterativeCsvWriter is designed to work with iterators that operate over query result sets. We currently support only java.sql.ResultSet... which may be all that we need ever to support. Files written with this writer should work with Vine.readFromCsvFile().
