Package: carnival.util

[Groovy] Interface MappedDataInterface


  • interface MappedDataInterface

    MappedDataInterface is a bridge between CompoundFeature and DataTable. The purpose of this interface is to retro-fit legacy vine methods so they work with both CompoundFeatures and DataTables. It is expected that this interface will be deprecated after all legacy vine methods have been migrated to the new VineMethod construction.
