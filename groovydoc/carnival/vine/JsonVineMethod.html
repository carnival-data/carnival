Package: carnival.vine

[Groovy] Class JsonVineMethod<T>

      • carnival.vine.JsonVineMethod 

      • @groovy.util.logging.Slf4j
abstract class JsonVineMethod<T>
extends VineMethod

        JsonVineMethod is a parameterized abstract that concretizes the call(...) methods of VineMethod where the data representation format is JSON, such as data returned from a REST API call. The fetch() method is left un-implemented. It is expected that client code will extend this class to implement their fetch business logic.
