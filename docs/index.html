Carnival builds upon a property graph database to empower data aggregation and harmonization using a novel set of graph building and data modelling tools.

DOCUMENTATION

Find documentation on Carnival.

The Carnival application programmer interface (API) is documented on a searchable website.

PROPERTY GRAPHS

Property graphs are the foundation of Carnival.

Carnival uses an embedded property graph to store data, which can be either Neo4j or an in-memory Tinkergraph from Tinkerpop.
