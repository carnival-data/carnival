Package: carnival.util

[Groovy] Class DataTable

    • carnival.util.DataTable 

    • abstract class DataTable
extends java.lang.Object

      DataTable is a generalized superclass that provides a basic framework for tabular data structure that can be written and read from disk. The core data structure of DataTable is not specified and is left to implementing sub-classes. DataTables can be written to and read from files. A single DataTable is represented by two files, a .csv file for the data and a.yaml file for the other bits of information that describe the data. The DataTable object provides a framework for keeping track of the files to which it has been written and from which it has been read.
