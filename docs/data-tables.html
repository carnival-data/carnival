Data Tables

Carnival contains a DataTable class with two sub-classes, MappedDataTable and GenericDataTable. Please see an example below about the basics of a MappedDataTable.

@Grab(group='edu.upenn.pmbb', module='carnival-util', version='0.2.6') (1)

import carnival.util.MappedDataTable
import carnival.util.KeyType

def mdt = new MappedDataTable(
    name:"myMappedDataTable",
    idFieldName:'ID' (2)
)

mdt.dataAdd(ID:'1A', NAME:'alex')
def currentDir = new File(System.getProperty("user.dir"))
mdt.writeFiles(currentDir)
1 Use @Grab to incorporate the carnival-util dependency.
2 Set the name of the identifier field of this data table to 'ID'
