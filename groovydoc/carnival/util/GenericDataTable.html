Package: carnival.util

[Groovy] Class GenericDataTable

      • carnival.util.GenericDataTable 

      • @groovy.transform.ToString(excludes: [data], includeNames: true)
class GenericDataTable
extends DataTable

        GenericDataTable is a semi-generalized structure storing key/value data associated with a single entity that has a unique identifier. GenericDataTables can be written to and read from files. A single GenericDataTable is represented by two files, a .csv file for the data and a .yaml file for the other bits of information. The GenericDataTable object itself keeps track of which files to which it has been written and from which it has been read.
