Package: carnival.vine

[Groovy] Trait Vine

      • carnival.vine.Vine 

      • @groovy.transform.Trait
trait Vine
extends MethodsHolder

        Vine is is a trait that can be applied to a class to enable the implementation of vine methods. The Vine trait is not required to implement vine methods. It provides a convenient mechanism to aggregate related vine methods and share common resources.

          • java.lang.Object methodMissing(java.lang.String name, java.lang.Object args)

            methodMissing() is a Groovy language feature that catches unknown method calls. Groovy metaprogramming Carnival uses methodMissing() to support calling vine methods with convenience methods. Ex. myVineObject.myFunMethod(a:1) would call the vine method named "MyFunMethod" with an argument named "a" with value 1.

            Parameters:
            name - The name of the missing method.
            args - The provided arguments.
